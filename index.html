Description

This plugin try to limit the value of input fields to numeric only values. It has some default configuration that can be overwritten.

Default configuration:

  • value used for increasing/decreasing the value of input default: 1
  • boolean indicating if the number can be float default: true
  • minimum value accepted default: null
  • maximum value accepted default: null
  • the default value of the input default: 0

Usage

<script type="text/javascript">
    $(function(){
        $("#ex3").NumericOnly({
            minValue: -15,
            maxValue: 12.5,
            float: false,
            step: 2.38,
            value: 3.2
        });
    });
</script>
